### 2. 💅 Make It Beautiful
An upgraded version of the profile page, styled with custom CSS. Features a dark hero banner, CSS variables for theming, a two-column hobbies grid, styled social link buttons, and a fully responsive layout for mobile screens.

**Tech:** HTML, CSS

---

### 3. 🖥️ Responsive Hero Page
A landing page with a navbar and a two-column hero section (text + image) built using Flexbox. Collapses gracefully into a single-column layout on smaller screens.

**Tech:** HTML, CSS

---

### 4. 🧮 Console Calculator
A JavaScript exercise that prompts the user for two numbers and logs five arithmetic results — addition, subtraction, multiplication, division, and modulus — to the browser console.

**Tech:** HTML, JavaScript

---

### 5. 📋 Interactive Form
A student registration form with live client-side validation. Checks for empty fields, a valid email format, and a minimum password length before showing a success or error message.

**Tech:** HTML, CSS, JavaScript

---

### 6. 🎓 Student Manager
A plain JavaScript script that stores student data (names and marks) in an array of objects, calculates the average marks for each student, and prints the results to the console.

**Tech:** JavaScript

---

### 7. 🌐 Hello Server
A minimal Node.js + Express server with three routes: Home (/), About (/about), and Contact (/contact). Serves a plain-text response for each route.

**Tech:** Node.js, Express

---

### 8. ✅ Dynamic List App
A React task manager where users can type a task, click Add to append it to the list, and click Delete to remove individual items. Built with useState for managing the task array.

**Tech:** React

---

### 9. 😊 Mood Tracker
A React app with three mood buttons (Happy, Sad, Angry). Clicking a button updates a message and dynamically changes the page background colour.

**Tech:** React

---

### 10. 🛍️ Product Listing UI
A mini e-commerce product listing page with search, max-price filter, and category filter buttons. Products are fetched from a hardcoded array; filtered results update instantly. Users can add items to a cart counter.

**Tech:** React

---

### 11. 🗺️ Multi-Page App
A multi-page personal portfolio SPA with five routes — Home, About, Hobbies, Subjects, and Contact — built using React Router v7. Each page is a separate component with fade-in animation.

**Tech:** React, React Router DOM

---

### 12. 🔐 Smart Signup Form
A two-page React app with a signup form and a dashboard. The form validates email format, minimum password length, and password complexity before navigating to the dashboard.

**Tech:** React, React Router DOM

---

### 13. 🌤️ Weather Dashboard
A React app that fetches live weather data from the OpenWeatherMap API. Users can search for a city and view temperature, weather condition, and humidity. Handles loading and error states.

**Tech:** React, OpenWeatherMap API
